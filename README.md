# OrbitBin – Geo Tracking of Waste Management System

## Overview
OrbitBin is a smart waste management system that combines Artificial Intelligence (AI), GPS-based geo-tracking, and web technologies to improve waste collection efficiency. The system enables users to report waste by uploading images, automatically detects waste using an AI model, captures the location, and notifies waste collectors for timely cleanup. It also provides dashboards for monitoring reports, analytics, and route optimization. :contentReference[oaicite:0]{index=0}

## System Workflow

1. User logs into the application.
2. User uploads an image of waste.
3. GPS captures the user's location automatically.
4. AI model analyzes the uploaded image.
5. Waste information is stored in the MySQL database.
6. Notification is sent to the nearest waste collector.
7. Staff monitors reports through the dashboard.
8. Route optimization helps collectors reach the waste location efficiently. :contentReference[oaicite:1]{index=1}

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/orbitbin.git
```

2. Move to the project folder.

```bash
cd orbitbin
```

3. Install frontend dependencies.

```bash
npm install
```

4. Configure the MySQL database.

5. Import the SQL file into phpMyAdmin.

6. Start the PHP server using XAMPP.

7. Run the Angular application.

```bash
ng serve
```

8. Open your browser.

```
http://localhost:4200
```
